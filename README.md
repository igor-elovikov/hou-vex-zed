# Houdini VEX for Zed

Zed extension providing language support for [Houdini VEX](https://www.sidefx.com/docs/houdini/vex/index.html).

## Features

- Syntax highlighting via [Tree-sitter grammar](https://github.com/igor-elovikov/tree-sitter-vex)
- Code snippets for built-in VEX functions
- Bracket matching and auto-indentation

## Supported File Types

`.vex`, `.vfl`
